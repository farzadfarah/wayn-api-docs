# WAYN API Documentation Portal

A ReadMe.io-style developer portal built with Next.js, React, TypeScript,
Tailwind CSS, shadcn/ui-style components, and Scalar for the API reference.

## Run Locally

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Build

```bash
npm run build
npm run start
```

## OpenAPI Source

Scalar loads the API definition from:

```text
/openapi.yaml
```

The source file lives at `public/openapi.yaml`.
